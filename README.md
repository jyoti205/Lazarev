# LAZAREV Clone

A modern clone of the Lazarev website built using HTML, CSS, JavaScript, GSAP, and ScrollTrigger.

##  Features

- Smooth GSAP animations
- Scroll-based animations using ScrollTrigger
- Interactive navigation menu
- Custom cursor effect
- Hover animations
- Lazarev-inspired UI and animations

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger

## ⚠️ Note

This project is currently optimized for desktop screens only. Responsive design will be added in future updates.

##  Future Improvements

- Make the website fully responsive
- Add more advanced animations
- Improve accessibility and performance
